## 部署

### 1. 下载
```bash
git clone https://github.com/truda8/upload-labs-plus.git
```

### 2. 进入目录
```bash
cd upload-labs-plus
```

### 3. 使用 docker-compose 构建
```bash
docker-compose up -d
```

### 4. 停止
```bash
docker-compose down
```
